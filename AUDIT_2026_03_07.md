### Vulnerabilities Found

#### ISSUE 1: XSS in PDF Export — Severity: HIGH

**File:** `board-studio/src/components/boards/ExportDialog.tsx` (lines 169-174, 317-354)

```typescript
const win = window.open('', '_blank');
if (win) {
  win.document.write(html);  // HTML generated without escaping
  win.document.close();
}
```

The `generatePdfHtml()` function concatenates user data directly into HTML:
- `<h1>${board.name}</h1>` — board name not escaped
- `<td>${c.name}</td>` — card names not escaped
- `<td>${g.name}</td>` — gate names not escaped

If board/card/gate names contain `<script>` tags or event handlers, they will execute.

**Fix:** Add HTML escaping utility:
```typescript
function escapeHtml(text: string): string {
  const div = document.createElement('div');
  div.textContent = text;
  return div.innerHTML;
}
```

#### ISSUE 2: Open Redirect Risk — Severity: MEDIUM

7 locations use `window.open()` with dynamically constructed URLs without origin validation:

| File | Pattern |
|------|---------|
| `workflow-studio/src/components/runs/RunArtifactsPanel.tsx` | `window.open(download_url, '_blank')` — URL from API response |
| `workflow-studio/src/components/artifacts/ArtifactCatalog.tsx` | Same pattern — API-provided URL |
| `board-studio/src/pages/WorkflowsPage.tsx` | `window.open(WORKFLOW_STUDIO_URL + path)` — env var + user IDs |
| `board-studio/src/pages/AgentsPage.tsx` | `window.open(AGENT_STUDIO_URL + path)` — env var + user IDs |
| `board-studio/src/components/boards/ExportDialog.tsx` | `window.open('', '_blank')` — opens blank window |
| `packages/shell/src/Header.tsx` | `window.open(url, '_self')` — URL not validated |
| `packages/shell/src/AppShell.tsx` | `window.open(url, '_self')` — URL from URL params |

**Fix:** Add URL validation helper that checks against allowed studio origins.

#### ISSUE 3: Missing CORS Configuration — Severity: LOW

All `vite.config.ts` proxy configs use basic setup without explicit CORS headers or credential handling:
```typescript
proxy: { '/v0': { target: 'http://localhost:8080', changeOrigin: true } }
```

#### Not Found (Good)

- No `eval()` or `Function()` usage
- No WebSocket connections without auth
- No sensitive data beyond tokens in localStorage
- No `.env` files committed

### Missing Vite Optimizations

All 3 `vite.config.ts` files lack:

- `rollupOptions.output.manualChunks` — no vendor chunk splitting
- `optimizeDeps.include` — no pre-bundling of common deps
- No explicit treeshake settings

**Recommended `manualChunks` config:**
```typescript
build: {
  rollupOptions: {
    output: {
      manualChunks: {
        'react-vendor': ['react', 'react-dom', 'react-router-dom'],
        'state': ['zustand', '@tanstack/react-query'],
        'charts': ['recharts'],
        'ui': ['@airaie/ui', '@airaie/shell', 'lucide-react']
      }
    }
  }
}
```
